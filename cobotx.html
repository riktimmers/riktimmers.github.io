Cobotx Work Experience

Programming and User Interface

At the beginning Cobots where programmed using their own scripting languages running on the Teach Pendant. Because of limitations with the User Interface and IOs for controlling peripherals we switched to using an IPC running Ubuntu, such that the Cobot can be programmed using c++ and a User Interface with Flutter/Dart.
The User Interface allows control for one or more cobots, show Safety Scanner status, allow product selection and show other information that is required.
For controlling conveyors, pushers and reading out sensors either a PLC was used, or the IPC was used in combination with a Modbus TCP Remote IO module.
